What's New

Trend Micro Worry-Free Business Security Services provides the following new features and enhancements.

Enhanced Scalability

Better control of information and notifications. New search functionality to specific troubled clients and new quick task functionality that  allows quicker operations of the user interface.

Console and License Self-Merge capability

New license stacking and console merge capability. Ability to add new license keys to an existing to increase your license count.

Improved Dashboard and Reporting

Configurable alert templates allow administrators to set which information should be presented in the system alerts they receive.

Improved Scalability and Usability

Support up to 10,000 connected computers. Improvements in the management interface result in the ability for all administrators to search, sort and view protected machines in a more productive fashion.

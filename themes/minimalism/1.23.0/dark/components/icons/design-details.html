Icon gird

Design icons on a simple geometric grid to ensure icon consistency and visual balance. Use a 16-pixel grid to design a system icon set.

Icon shapes

To create icons on a gird, use basic shapes (such as circles, horizontal and vertical rectangles, and squares). Ensure that you use the same basic shape to maintain consistent proportions when designing an icon set.
The following shows the dimensions of sample 16-pixel icons on a 16-pixel grid:

  • Circles: 16 x 16 px
  • Squares: 14 x 14 px
  • Horizontal rectangles: 12 x 16 px
  • Vertical rectangles: 16 x 12 px

The following are the dimensions of sample 32-pixel icons on a 32-pixel grid:

  • Circles: 32 x 32 px
  • Squares: 28 x 28 px
  • Horizontal rectangles: 24 x 32 px
  • Vertical rectangles: 32 x 24 px
Line weight

16-pixel icon is the default small icon size. The normal line weight or thickness is 1px. Depending on the design object, you can set the line weight to 2px for emphasis.
For 32-pixel icons, increase the normal line weight to 2px.

Icon colors

Use icon font to set the colors for system icons.
Use gray (#BBBBBB) for normal display and white (#FFFFFF) for rollover.

Border radius

Specify the outside border radius to create rounded border corners for user-friendly appearance and design consistency.

  1. Set outside border radius to 1px for 16-pixel icons, or to 2px for 32-pixel icons.
  2. Do not set the inside border radius. The inner corner should be a square.
  3. Do not set the radius for the end of a line.
Icon sizes

Design icons with equal width and height (for example, 16 x 16 px) to prevent distortion.
For consistency and ease of maintenance, create icons based on common sizes (for example, 16 x 16 px, 24 x 24 px, 32 x 32 px). You may design icons of other sizes depending on the product specifications.

Format

For system icons, use Icon Font. Use the SVG format if an icon is bigger than 32 pixels or contains more than one color.
If your product supports older web browser versions (for example, Internet Explorer 8 or earlier), provide PNG files.

Switch container

Wrap an input and a div within an element, such as label.
Checkbox class

Use the checkbox class on an input element, and the type attribute is checkbox.
Toggle class

Use the toggle class on a div element.
Toggle button class

Add a pseudo-element, for example, .toggle:before, on the toggle class.

A toggle switch allows the user to indicate a state of on vs. off or yes vs. no.

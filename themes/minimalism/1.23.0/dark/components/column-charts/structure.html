Column charts use vertical bars to present and compare data over time or over values containing an intrinsic order. The best way to represent these values is to use the horizontal axis.

  • Use qualitative colors to display your data. Use the same color for data sets of the same variable.
  • Ensure the category has an intrinsic order. If not, consider using a bar chart.
  • Avoid displaying labels at 45° or 90° on the axes. This makes labels difficult to read.
  • Shorten well-known labels to improve readability. For example, use "Jan" for "January".
  • Avoid using gradient colors and 3-D bars.

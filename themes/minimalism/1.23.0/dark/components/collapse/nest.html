Nest collapsible panels

Extend the default panel styles for Nest Collapsible Panels.

For a large data set or complex information, you may nest collapsible panels within collapsible panels to display structured content.

  • When a panel is in the "collapse" state, change the background color and the pointer cursor of the collapsible group item when the user hovers over any part of that row.
  • When a panel is in the "expand" state, change the pointer cursor of the collapsible group item when the user hovers over any part of that row.
  • Bold the group title in expanded state to differentiate the group title from the expanded content.

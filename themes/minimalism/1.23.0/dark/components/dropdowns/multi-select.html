Wrap an and a within a element to implement a check box for each drop-down menu item. Please refer to Form Elements.

You can use check boxes in drop-down menus to indicate item selection. As a summary, you can display a count for the number of selected items or list them out. Separate the listed items with commas, and put an ellipsis (...) at the end to indicate omission if the list is too long to fit into one line.

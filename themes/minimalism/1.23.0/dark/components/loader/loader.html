CSS3 spinner

The animated progress icons are created using CSS3 keygrames.
Loader class

Use the loader class on a element. For example, .loader
Loader overlay class

Wrap the loader within a element. For example, .loader-overlay

Use a loader to indicate progress or an indeterminate amount of time required for a process.

  • Loader size: Loaders can have different sizes (e.g. small, medium, or large), depending on your requirements.
  • Loader overlay: Sometimes, it can be useful to overlay the loader and temporarily block access to other functionalities on the UI.

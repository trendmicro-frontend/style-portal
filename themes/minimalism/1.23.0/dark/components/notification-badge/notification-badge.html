Use a notification badge to display reminders and alert information, and notify users of unread messages.

A notification badge consists of the following elements:

  • Numeric badge: Displays the number of new notifications. Click the numeric badge to display or hide the popover window.
  • Popover window: Displays new notifications for a user.

A notification badge:

  • Should be located on the top-right of a banner.
  • Display an updated counter when a new notification becomes available.
  • Resets the counter and hides the numeric badge after displaying the popover window.
  • Includes a red dot in front of an unread notification.
  • Displays an empty popover window to indicate no new notification is available.
  • Displays a loader icon to indicate that data retrieval is in progress.

Regular Search allows users to query data by doing one of the following actions:

  • Click the search button
  • Click the reset button
  • Press "Enter" on the keyboard

It is recommended to use the default button style for the search button. You may also use the primary or emphasis style.

You can include a reset button to clear the search input field and trigger a new search. Only display the reset button when the input field contains a value.

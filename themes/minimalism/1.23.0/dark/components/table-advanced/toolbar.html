A table toolbar contains control buttons applicable to the table. Make sure these controls are workable when multiple items are selected.

  • Put table toolbars on top of the tables. Use different background color for your toolbars depending on the table type or specific needs of the product.
  • If your users use toolbars very often, consider fixing the table toolbar while the table is being scrolled down.
  • Use primary buttons on toolbars only to highlight the most important button.
  • You can build constraints on your toolbars where you only display buttons that are applicable or accessible for the current configuration or item selection.

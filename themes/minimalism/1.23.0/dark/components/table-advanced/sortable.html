Sortable table headers

Entries within a column can be sorted by clicking their column header.

  • The first click should sort entries in descending order, and the second click in ascending order.
  • When such headers are hovered over, display a pointer and show the header in another color to indicate sortability.
  • The sort icon should be visible on the right in a sortable column, regardless of the column width.

When two or more entries contain the same value after sorting a table based on one column, users can sort the entries again based on another column.

  • When a table is sorted based on more than one column, display a number (as superscript) in the column header to indicate the sorting priority.
  • To indicate that a table column is sortable, do the following when the cursor hovers over the column header: display a drop-down icon, display a pointer, and change the header color.
  • Users can sort a column by clicking the column header or selecting the sorting criteria from the drop-down list.
  • For a sorted column, click the column header to change the sorting direction (for example, from ascending sort to descending sort).
  • Press CTRL+click on a column header to specify a second sorting criteria.

Banner notifications are messages about banner statuses and events that require user attention and sometimes actions. A banner notification:

  • should appear on the top of a product console, just below the navigation bar.
  • should be full-width, static, and closable.
  • can contain hyperlinks that redirect users to another UI screen or action buttons to perform a task.
  • involving critical issues should only be closable when the alerted situations are resolved.

Types:

  • Error: This type of banner notification is used to notify critical issues that require the user's immediate attention and action.
  • Warning: This type of banner notification is used to notify potential issues, but users may not need to do anything.
  • Informational: This type of banner notification is used to provide users with potentially useful, relevant information.

For detailed information, see Banner Messages.

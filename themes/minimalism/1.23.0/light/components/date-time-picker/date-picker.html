The date picker contains the date input field and the calender view. Users can either use the input field to type a date or by selecting a date in the calendar.

  • Click the input field to open the calendar dropdown.
  • Click the arrows to view the previous and next days, months, or years.
  • Click a certain month (or year) to show all the days (or months) of that month (or year).
  • A selected date is marked with a red circle and the numeral of the current date is shown in red.
  • If no default date is selected, show a placeholder help text to indicate the date format.
  • As soon as a date is selected, the calendar closes itself and the selected date is shown in the input field.
  • As soon as a date is typed, the date is also circled on the calendar.
  • Support selecting using keyboards.
  • Date formats may be adjusted as required. For detailed information about different date format variations, see Date & Time.

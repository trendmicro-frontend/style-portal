Inline form class

Use a form class on a block-level element. For example, .form-inline is used as an inline form class in Bootstrap.

Inline form group class

Use a block-level element to wrap form control and labels. For example, .form-group is used as a form group class in Bootstrap.

Inputs with form control

For details on styles and statuses of inputs, refer to Form Elements.

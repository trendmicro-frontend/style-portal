Light collapsible panels

Extend the default panel styles for Light Collapsible Panels.

Light collapsible panels can be used to reduce visual clutter.

  • When a panel is in the "collapse" state, change the background color and the pointer cursor of the collapsible group item when the user hovers over any part of that row.
  • When a panel is in the "expand" state, change the pointer cursor of the collapsible group item when the user hovers over any part of that row.
  • Bold the group title in expand state to differentiate the group title from the expanded content.

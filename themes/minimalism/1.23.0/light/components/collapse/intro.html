Collapsible Panels

Use to create a panel class. For example, .panel is used in Bootstrap.

Collapsible panels provide users with the ability to expand and collapse content as needed, they allow you to hide content that is not immediately relevant to the user. It can operate as either a single collapsible element (similar to an accordion) or multi-collapsible element.

  • Single collapsible element: Allow only one panel to be expanded at any given time. Expanding a panel collapses the previously expanded panel.
  • Multi-collapsible element: Allow more than one panel to be expanded at any given time. Expanding a panel has no impact on the other panels.

Follow these rules when designing charts:

  • Only use colors from the above color palettes to ensure consistent visual design.
  • Do not use qualitative and sequentical colors in the same chart.
  • Always use Color 1 as the first color when you need a set of qualitative colors.
  • If you have 5 categories or less, select colors from the qualitative colors that match your purpose.
  • If you have more than 5 categories, use qualitative colors in the recommended order.
  • If you have 11 categories or more, apply the qualitative colors again in the recommended order.

Depending on your display language, specify the following fonts in your font-family. These are options that can be used when a particular font is not available. For example, an interstate-related font should be used to display English text, and when such font is not available, Arial should be used.

  • ZH-TW: "微軟正黑體","新細明體","細明體"
  • ZH-CN: "微软雅黑","宋体"
  • JA-JP: 'ＭＳＰゴシック', 'MS PGothic', 'メイリオ', Meiryo
  • KO-KR: "Dotum"
  • TH-TH: Tahoma

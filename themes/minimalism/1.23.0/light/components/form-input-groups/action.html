Extend the input by adding buttons, drop-down menus, or both to the front, the end, or both sides of a search box to guide users' search action.

  • Use the default button style for the action button, which is the most important button in an input group. You may also use the primary or emphasis style for emphasis.
  • If a drop-down menu exists to the front of the search box, and an option has been specified, the input should be focused.

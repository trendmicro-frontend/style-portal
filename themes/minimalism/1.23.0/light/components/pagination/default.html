Pagination class

To paginate, use a pagination class on a element. For example, .pagination is used as a Bootstrap pagination class.
Pagination items

Put under the element.
Pagination text

Use for pagination text under a element.
Pagination items state

Use the activated state to indicate the current selected page. For example, .active is used in Bootstrap. Use the disabled state to indicate that the item is inaccessible. For example, .disabled is used in Bootstrap.
  • Separate your content to different pages for better system performance when it is displayed.
  • Always put page numbers at the bottom of the page.
  • Provide navigations for previous or next pages.

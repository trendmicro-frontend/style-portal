Tables with draggable columns allow users to rearrange the order of table columns.

  • Only allow one column to be dragged at a time.
  • Click and drag a column header to the desired position.
  • When mouseover a draggable column header, display a visual hint (a pointer cursor).
  • When dragging a column header, display a visual hint (a move cursor).
  • Display a visual hint (change font and background colors in the column header) to indicate where a dragged column will be dropped.
  • Selection columns (for example, checkboxes or radio buttons) are not draggable.

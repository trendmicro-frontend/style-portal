Tables with draggable rows allow users to rearrange the order of records within tables.

  • Only allow one row to be dragged at a time.
  • When mouseover a draggaboe row, display a visual hint (a draggable indicator icon).
  • When dragging a row or mouseover a draggable indicator icon, display a visual hint (a move cursor).
  • Display a visual hint to indicate where a dragged row will be dropped.
  • Drag-and-drop tables should have fewer than 50 rows and not display pagination controls.

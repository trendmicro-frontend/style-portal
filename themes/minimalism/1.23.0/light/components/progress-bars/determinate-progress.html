Progress class

Use a element to create a progress class. For example, .progress is a progress class used in Bootstrap.
Determinate indicator

Use a element wrapped within the progress class. Update its' width to indicate current progress.
Determinate indicator effect

Show a transparent overlap for the indicator effect by adding a pseudo-element, for example, :before, on the determinate indicator.
Label

A determinate indicator can contain a text value indicating current progress.

Use a determinate progress bar for operations where progress can be accurately calculated. It helps to quickly give a sense of how long an operation will take.

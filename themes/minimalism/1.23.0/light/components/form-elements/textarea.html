Rows

By default, a textarea must be able to contain at least two rows of text. Specify the textarea using the cols and rows attributes.

A textarea allows the user to type in extended, multi-line inputs.

A textarea can be fixed in height, grow automatically based on the input, or contain a resize handle, which allows for vertical resizing.

For a resizable textarea, use a fixed width.

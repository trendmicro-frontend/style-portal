Pie charts are used to compare values as proportions or percentages of a whole. The data should be categorical.

  • Start the pie chart at "12 o'clock" and move in a clockwise direction.
  • Ensure that the segments are ordered by size (largest to smallest).
  • Limit slices to ten. The chart becomes illegible when there are too many values to be represented.
  • Avoid using color gradients and three-dimensional charts.
